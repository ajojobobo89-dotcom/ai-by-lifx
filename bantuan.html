📖 Panduan Penggunaan

Bagaimana cara memulai chat? ▼

1. Ketik pertanyaan di kolom chat

2. Tekan Enter atau klik tombol Kirim

3. AI akan merespon dengan jawaban dari database manual

4. Anda juga bisa klik saran pertanyaan yang tersedia

Apakah bisa pakai singkatan? ▼

BISA! AI akan mendeteksi singkatan umum:

  • hlo, hi → halo/hai
  • mksh, thx → terima kasih
  • maaf, mf → maaf
  • iya, y, ya → iya
  • ga, gk, tdk → tidak
Apakah butuh API key / internet? ▼

TIDAK PERLU! AI ini 100% offline:

✅ Tidak perlu API key

✅ Tidak perlu koneksi internet

✅ Semua respons ditulis manual

✅ Aman & privasi terjaga

Berapa banyak respons yang tersedia? ▼

Saat ini tersedia 2,047 respons dalam 48 kategori:

📚 Pendidikan: 203 respons

💻 Teknologi: 156 respons

🏥 Kesehatan: 145 respons

➕ Matematika: 134 respons

Dan masih banyak lagi!

Bagaimana cara menambah respons? ▼

Anda bisa menambah respons dengan:

1. Buka file js/responses-2000.js

2. Tambahkan di kategori yang sesuai:

{
    keywords: ["pertanyaan", "versi singkat"],
    response: "Jawaban yang diinginkan"
}

3. Simpan file dan refresh browser

  • Statistik penggunaan

  • Reset percakapan
  • Dark/light theme
  • 100% offline tanpa API key
