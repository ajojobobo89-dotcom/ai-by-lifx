## 📋 Deskripsi

**AI Asisten Manual** adalah asisten virtual berbasis web yang **100% GRATIS** dan **TANPA API KEY**. Semua respons ditulis manual oleh developer, sehingga tidak memerlukan koneksi internet ke server eksternal. Cocok untuk belajar, portofolio, atau kebutuhan chat internal.

### ✨ Fitur Utama

- ✅ **2000+ Respons Manual** - Database lengkap berbagai kategori
- ✅ **48 Kategori** - Dari teknologi hingga motivasi
- ✅ **Deteksi Singkatan** - "hlo" otomatis dibaca "halo"
- ✅ **100% Offline** - Tidak perlu internet setelah dimuat
- ✅ **Tanpa API Key** - Benar-benar gratis selamanya
- ✅ **Multi Halaman** - Dashboard, kategori, statistik, pengaturan
- ✅ **Export Chat** - Simpan percakapan ke file
- ✅ **Responsive Design** - Bisa diakses dari HP/tablet/desktop
